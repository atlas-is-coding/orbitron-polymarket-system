# i18n Design: Multi-Language Support

**Date:** 2026-02-25
**Status:** Approved

## Requirements

- Languages: EN, RU, ZH, JA, KO
- Language stored in `config.toml` under `[ui]`
- Translate everything: TUI, logs, Telegram notifications
- Hot-reload: language switch takes effect immediately without restart or save
- Translation files: JSON embedded in binary via `//go:embed`

## Architecture

### Package `internal/i18n/`

```
internal/i18n/
  locale.go      — Locale struct (all translatable strings)
  manager.go     — Manager, T(), SetLanguage(), atomic.Pointer[Locale]
  embed.go       — //go:embed locales/*.json
  locales/
    en.json
    ru.json
    zh.json
    ja.json
    ko.json
```

**`Manager`** with atomic pointer:

```go
type Manager struct {
    current atomic.Pointer[Locale]
    locales map[string]*Locale  // preloaded at startup
}

var global Manager

func T() *Locale           { return global.current.Load() }
func SetLanguage(lang string) { ... }  // atomic store, no locks
```

### Config Changes

New `[ui]` section in `config.toml`:

```toml
[ui]
language = "en"  # en | ru | zh | ja | ko
```

New `UIConfig` in `internal/config/config.go`:

```go
type UIConfig struct {
    Language string `toml:"language"`
}

type Config struct {
    // ...existing fields...
    UI UIConfig `toml:"ui"`
}
```

Startup sequence in `main.go`:

```go
cfg, _ := config.Load(cfgPath)
i18n.SetLanguage(cfg.UI.Language)  // before TUI starts
```

### Hot-Reload Mechanism

`FieldDef` gains an optional `OnChange` callback:

```go
type FieldDef struct {
    // ...existing fields...
    OnChange func(v string)  // called on every toggle, before save
}
```

Language field in `allFields`:

```go
{
    Section: i18n.T().SectionUI, Label: ..., Kind: KindEnum,
    Options: []string{"en", "ru", "zh", "ja", "ko"},
    Get:      func(c *config.Config) string { return c.UI.Language },
    Set:      func(c *config.Config, v string) error { c.UI.Language = v; return nil },
    OnChange: func(v string) { i18n.SetLanguage(v) },
}
```

In `settings.Update()`, KindEnum toggle calls `OnChange` immediately:

```go
case KindEnum:
    m.optionIdx[m.cursor] = (m.optionIdx[m.cursor] + 1) % len(f.Options)
    if f.OnChange != nil {
        f.OnChange(f.Options[m.optionIdx[m.cursor]])
    }
```

BubbleTea re-renders after every `Update()` → next `View()` cycle picks up the new locale via `i18n.T()`.

### Data Flow

```
i18n.SetLanguage("zh")        ← Settings OnChange / main.go startup
    ↓
atomic.Pointer[Locale].Store  ← thread-safe, no mutexes
    ↓
i18n.T()                      ← any goroutine, anywhere
    ├── TUI View()             → instant hot-reload
    ├── zerolog .Msg()         → next log entry
    └── Telegram Send()        → next alert
```

### Log Integration

```go
// Before:
log.Info().Str("order_id", id).Msg("Order filled")

// After:
log.Info().Str("order_id", id).Msgf(i18n.T().LogOrderFilled, id, price)
```

### Telegram Integration

```go
// Before:
msg := fmt.Sprintf("✅ Order filled: %s\nPrice: %.4f", id, price)

// After:
msg := fmt.Sprintf(i18n.T().TgOrderFilled, id, price)
```
